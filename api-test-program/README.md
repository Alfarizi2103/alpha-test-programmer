## Cara Menjalankan Program
- git clone <url repository>
- composer install
- composer update
- copy .env.example dan hasil copy rename menjadi .env
- php artisan key:generate
- set configurasi database pada .env
- php artisan migrate
- php artisan serve
- tes api pada github
